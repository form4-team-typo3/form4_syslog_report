# form4 SysLog Report

## Introduction

This TYPO3 extension sends **e-mail reports** for `sys_log` (system log) entries,
using a **scheduler task** or the **CLI**.

![Scheduler task in the TYPO3 backend](Documentation/Images/Task.png)
